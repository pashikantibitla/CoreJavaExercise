### 5.3 Attempt 3: Pandoc HTML + Playwright PDF

**Step 1: Convert Markdown to HTML**
```
pandoc -f markdown -t html -o "D:\Learning_Playwright_with_javascript\Java_Selenium_Guide.html" "D:\Learning_Playwright_with_javascript\Java_Selenium_Guide.md"
```
- Result: SUCCESS
- HTML file created at: `D:\Learning_Playwright_with_javascript\Java_Selenium_Guide.html`

**Step 2: Convert HTML to PDF using Playwright**
```python
from playwright.sync_api import sync_playwright

html_path = r'D:\Learning_Playwright_with_javascript\Java_Selenium_Guide.html'
pdf_path = r'D:\Learning_Playwright_with_javascript\Java_Selenium_Complete_Guide.pdf'

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto('file:///' + html_path.replace('\\', '/'))
    page.pdf(
        path=pdf_path,
        format='A4',
        margin={'top': '40px', 'bottom': '40px', 'left': '40px', 'right': '40px'}
    )
    browser.close()
```

**Result:** SUCCESS
- PDF generated at: `D:\Learning_Playwright_with_javascript\Java_Selenium_Complete_Guide.pdf`
- Format: A4 with 40px margins on all sides
- Renderer: Chromium (via Playwright)
