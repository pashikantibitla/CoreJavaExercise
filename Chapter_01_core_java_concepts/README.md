## 📋 Key Takeaways

1. **Java is platform-independent** — Write Once, Run Anywhere (WORA) via JVM.
2. **Strongly typed language** — Every variable must have a declared type.
3. **No pointers** — Memory management is automatic (Garbage Collection).
4. **Pure OOP** — Everything is an object except primitives.
5. **Robust** — Exception handling, strong memory management, no pointers.
6. **Multithreaded** — Built-in support for concurrent programming.
7. **Identifiers** — Names given to classes, methods, variables (case-sensitive).
8. **Reserved words** — 53 keywords (including literals true, false, null).
9. **Data types** — 8 primitives + reference types (class, interface, array, enum).
10. **Arrays** — Fixed-size, homogeneous, indexed collection of elements.
11. **Variables** — Local, instance, static (class variables).
12. **Main method** — `public static void main(String[] args)` — JVM entry point.
13. **Operators** — 44 types of operators in Java with defined precedence.
14. **Flow control** — Sequential, conditional, iterative, transfer control.
15. **Access modifiers** — public, private, protected, default (package-private).
16. **OOPs** — Abstraction, Encapsulation, Inheritance, Polymorphism.
17. **Exception hierarchy** — Throwable → Exception (checked) / Error (unchecked).
18. **Threads** — Lightweight processes, multiple execution paths.
19. **Synchronization** — Thread-safe operations via locks and monitors.
20. **Inner classes** — Classes defined within other classes for better encapsulation.
