## Compilation & Execution

```bash
# Compile Java programs
mvn clean compile

# Run tests
mvn test

# Run specific test class
mvn test -Dtest=RestAssuredBasicTest

# Run with TestNG suite
mvn test -Dsuite=regression.xml

# Run Newman collection
newman run collection.json -e environment.json

# Run Newman with data-driven testing
newman run collection.json -d data.csv -n 5
```
