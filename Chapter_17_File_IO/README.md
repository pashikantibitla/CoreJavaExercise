## 🎯 Key Takeaways

1. **File** class represents file/directory paths and provides metadata operations.
2. **FileWriter/FileReader** are convenience classes for character-based file I/O.
3. **BufferedWriter/BufferedReader** add buffering for efficient reading and writing.
4. **PrintWriter** provides convenient methods for formatted output.
5. **InputStream/OutputStream** are abstract base classes for byte-level I/O.
6. **ObjectInputStream/ObjectOutputStream** enable Java object serialization.
7. **FileInputStream/FileOutputStream** read and write raw bytes to files.
8. **NIO.2** (Java 7+) provides `Path`, `Files`, and `Paths` for modern file I/O.
