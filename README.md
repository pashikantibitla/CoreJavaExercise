└── Generics

Phase 4: Collections & Data Structures

├── Concurrent Collections
└── String/StringBuffer/StringBuilder

Phase 5: JVM & Memory Management
├── JVM Architecture

├── File I/O & Serialization
├── Regular Expressions
├── Assertions
└── Internationalization

└── Development & Build Tools
